Interactive Machine Learning Models

Inception Model

This model of inception has been trained on millions of images and uses the "ImageNet" dataset to classify objects. Simply input a URL for a particular image, and the model will identify the top 10 different possibilities for the subject of the image along with its confidence percentage for each.

Github Repository: https://github.com/faas-and-furious/inception-function

Pigo Blur Model

The Pigo model identifies and blurs faces in images using facial recognition techniques in machine learning. It takes in a URL of an image and returns the photo but blurred for any faces that it is able to detect. Note that the photo must clearly show a face or multiple faces, or else it may fail.

Github Repository: https://github.com/esimov/pigo-openfaas-faceblur

Colidr Model

The Colidr model recreates an image with black lines to give the appearance of a hand-sketched drawing. It is best to use sharp images with clearly defined edges and features to accurately sketch a copy of it.

Github Repository: https://github.com/esimov/openfaas-coherent-line-drawing

Colorization Model

The Colorization function can turn any black-and-white photo into a fully colored image. The black-and-white input must be as a URL for the request, and the model may not work as intended for already colored images.

Github Repository: https://github.com/alexellis/repaint-the-past
